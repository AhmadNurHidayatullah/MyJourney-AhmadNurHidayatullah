Projects

Login App

Aplikasi web ini merupakan sistem autentikasi sederhana pengguna yang memungkinkan proses login dan logout dengan penyimpanan data berbasis database. Aplikasi dibangun menggunakan Spring Boot sebagai backend dan Thymeleaf untuk antarmuka, serta terintegrasi dengan MySQL dari XAMPP untuk menyimpan data pengguna.

Web Book Store

Web Book Store adalah aplikasi web sederhana yang terdiri dari halaman Home yang berisi sambutan, sejarah munculnya toko buku, serta informasi kantor cabang; halaman About yang menjelaskan berbagai genre dan informasi tentang buku; serta halaman Contact yang menyediakan kotak saran dan peta lokasi.

Input Data

Aplikasi ini merupakan sistem web sederhana yang memungkinkan pengguna untuk menginput data melalui form dan melihat hasil data yang sudah diinput dalam bentuk daftar. Data yang diinput akan disimpan secara ke dalam database MySQL yang berjalan melalui XAMPP. Aplikasi ini cocok digunakan untuk keperluan pendataan seperti biodata, data barang, atau data lainnya.

sistem Pembelian Tiket

Sebuah web sederhana yang memungkinkan pengguna melihat daftar tim F1 yang bertanding, memesan tiket dengan mengisi form (nama, email, jumlah tiket, jenis tiket, dan layanan tambahan), serta mengakses informasi kontak dan media sosial penyelenggara. Dibuat dengan fokus pada kemudahan penggunaan.

Sistem Pendaftaran Karyawan

Web sederhana dengan satu halaman form pendaftaran karyawan yang berisi input nama, umur, email, dan bidang yang diminati. Dilengkapi validasi data dan alert notifikasi saat form berhasil dikirim atau terdapat kesalahan pengisian.

Web Sejarah debut Ferrari di F1

Web informatif yang membahas awal mula dan kesuksesan Ferrari di F1. Halaman utama menampilkan logo Ferrari beserta penjelasannya, halaman About berisi perjalanan Ferrari dari awal berdiri hingga debut dan kiprahnya di F1 pada era 1950–1980-an, serta halaman lain menampilkan video transisi perkembangan mobil Ferrari di F1 dari tahun ke tahun.

Pendataan Hewan

Aplikasi web sederhana yang digunakan untuk melakukan pendataan hewan. Terdiri dari satu halaman utama yang mendukung fitur CRUD (Create, Read, Update, Delete) data hewan. Fitur utama: Tambah, edit, dan hapus data hewan secara langsung dari tampilan tabel. Fitur pencarian berdasarkan nama dan spesies hewan untuk mempermudah pengelolaan data. Terhubung dengan database MySQL menggunakan XAMPP sebagai backend server lokal.

Web Album

Website sederhana yang menampilkan profil James Arthur, terdiri dari halaman Home dengan tulisan James Arthur, About yang berisi biografi singkat, Musik yang menampilkan daftar lagu-lagu populer, dan halaman Contact untuk memberikan feedback atau pertanyaan.
